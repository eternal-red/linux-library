Shortcuts

Terminal Hotkeys

ctrl+a Set cursor to start of line
ctrl+e Set cursor to end of line
ctrl+u Delete from start until cursor
ctrl+k Delete from cursor until end
ctrl+r Search history
ctrl+c Exit out of a CL script (for most scripts)

Special Characters

>: redirect the output to this file
~: shorthand for path to your home directory
whatis [command]- simple summary of a command
man [command]- detailed description of the command

Search

grep [switches] [search input] [file or folder]
description: locate strings in a file
switches
-i: ignore case
-r: allows you to search a directory
-n display line that is greped
-w: only returns whole word (where a whole word matches the search input)
-v: inverted search, prints all lines that don’t include the search input
-A[# of lines]: number of additional lines printed after match term
-B[#number of lines]: before
repetition expressions
? The preceding item is optional and matched at most once.
* The preceding item will be matched zero or more times.
+ The preceding item will be matched one or more times.
{n} The preceding item is matched exactly n times.
{n,} The preceding item is matched n or more times.
{,m} The preceding item is matched at most m times. This is a GNU extension.
{n,m} The preceding item is matched at least n times, but not more than m times.
examples:
grep -i “ca.” file.txt
grep -i “pyt?h?o?n?” file.txt
would find python and py
notes
if you actually want to grep for a repetition expression, you have to use a backspace before it
ex: grep -r “hello/?” file.txt
find [directory] [switches] –name [search term]
description: locate files or directories on a device
switches
-type [d/f]: specify if you’re looking for either directories or files
-iname: makes search case insensitive
-user [username]: specify owner of the file
-size [subswitch: specify size of the file
[-a number]: matches values less than number
[a number]: matches values exactly the size of number
[+a number]: matches values greater than number
[c]: suffix for bytes
[k]: suffix for kilobytes
[m]: suffix for megabytes
ex: [-28m]
-perm [sub switches]: specifies permissions of the file
[###]: permissions in octal form
ex: [644]
[u=rw]: permissions in symbolic form
[-]: prefix that will return files that match at least the perms specified
[/]: prefix that will return files that match any of the perms specified
ex: [/666]
-time related searches
[min]: time will be specified in minutes
[time]: time will be specified in days
[a]: when a file was last accessed
[m]: when a file was last modified
[c]: when the status of a file last changed
-the time value is specified similar to the size switch
ex: -mtime +28
