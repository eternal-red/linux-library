System Commands

Booting

systemctl reboot
restart your computer
switches
--force: forcequit out of applications then reboot instead of waiting for them to safely close
--force --force: hard reset, equivilant to hitting the reset button on computer
shutdown [time] '[message]'
schedule a time to shutdown your computer
switches
-r: reboot instead
'[message]': you can choose to display a message to anyone still using the computer shortly before rebooting or not
systemctl get-default
change default boot target
useful targets
multi-user.target
rescue.target
graphical.target
systemctl set-default [target boot]
change default boot target
systemctl isolate [target boot]
temporarily use another boot target
