System Commands

Booting

systemctl reboot
restart your computer
switches
--force: forcequit out of applications then reboot instead of waiting for them to safely close
--force --force: hard reset, equivilant to hitting the reset button on computer
shutdown [time] '[message]'
schedule a time to shutdown your computer
switches
-r: reboot instead
'[message]': you can choose to display a message to anyone still using the computer shortly before rebooting or not
systemctl get-default
change default boot target
useful targets
multi-user.target
rescue.target
graphical.target
systemctl set-default [target boot]
change default boot target
systemctl isolate [target boot]
temporarily use another boot target

Startup Processes

systemctl
can managae and configure services

systemctl start [service file] starts the service immediately
systemctl stop [service file] gracefully quits out of the service immediately
systemctl status [service file] see if the service is currently running and if it is configured to run when the computer starts up.
systemctl enable [service file] enable the service to run on startup
systemctl disable [service file] disable the service from running on startup
systemctl revert [service file] revert the configuration of the service file to default factory settings
