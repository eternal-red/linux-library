Networking

Routing

ifconfig
prints all network interfaces
arp
find devices on your subnet
switches
-a: list all devices on subnet (stored in the arp table)
-d [ip]: delete a specified device from arp table
route
print the routing table for your device
ip route add [subnet] via [router ip]
connects to other subnets in your network and allows you to connect with them
netstat [switches] list all port connections
switches:
-l: list only listening sockets
-n: show the port number
-t: list TCP ports
-u: list UDP ports
